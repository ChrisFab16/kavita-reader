# Quickstart: Kavita Personal Lists (phased)

Run against a Kavita server with mixed content: in-progress series, want-to-read entries, at least one reading list, one bookmark, one rated series, one collection.

## Phase 1 — On Deck + Want to Read

1. Home → **On Deck** shows in-progress series; tap resumes to series/reader.
2. Remove from deck → item disappears; confirm on Kavita web.
3. Series detail → toggle **Want to Read** → appears under Home → **Want to Read**.
4. Toggle off → removed on server after refresh.
5. **Collections** regression: still browsable.

## Phase 2 — Starred + Reading Lists

1. Set star rating on series in app → matches Kavita web.
2. Home → **Starred** shows rated series.
3. Home → **Reading Lists** → open list → tap item → correct chapter opens.

## Phase 3 — Bookmarks

1. In reader, bookmark page → indicator on.
2. Home → **Bookmarks** → tap entry → opens same page.
3. Unbookmark in app → removed from shelf and web.

Record in `validation-results.md`.
