# Quickstart: Landscape Interface

1. Open app on Android emulator or device.
2. **Home** — rotate landscape; library cards show 3+ columns, no overflow; **no search bar** until header magnify tapped.
3. Tap **magnify** (header) → search bar appears; filter libraries; tap **close** → bar hidden, query cleared.
4. **Library detail** — grid adds columns; sort chips visible; header search same pattern; collection hint hidden in landscape.
5. **Series detail** — header and chapter list usable; inline search works (unchanged from 010).
6. **Settings** — scroll and toggle a setting in landscape.
7. **Image reader** — rotate in reader; zoom/pan still work; exit reader — app stays landscape.
8. **EPUB reader** (if available) — rotate; page turn zones and chrome OK.

Record pass/fail in `validation-results.md`.
