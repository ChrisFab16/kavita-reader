# Spec Kit Features

Feature specifications live in numbered directories: `specs/001-<short-name>/`, `specs/002-<short-name>/`, …

Each feature typically includes:

| Artifact | Purpose |
|----------|---------|
| `spec.md` | Requirements and user stories |
| `plan.md` | Technical approach and architecture |
| `tasks.md` | Implementation checklist |
| `quickstart.md` | Manual validation steps |
| `checklists/` | Quality gates |

Create a new feature with `/speckit-specify` or the Spec Kit scripts in `.specify/scripts/`.

Active feature path is recorded in `.specify/feature.json`.
